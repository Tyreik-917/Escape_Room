## Project Description
Its a Puzzle Escape room. 4 Levels, Each level has puzzles that need to be completed in order to go on to the next level.
**RESOLUTION NEEDS TO BE SET TO 1920 BY 1080**
***

## Program Design
### Features
1. Player Movement with Animations
2. Interact with items that have special functions
3. By completion of a level you move on to the next one (until level 4)
4. Menu Screens (Start, Pause, End)
### Game Stages
1. Start Menu
2. Level 1 (Escape Attic) 
3. Level 2: (Escape Bedroom)
4. Level 3 (Escape Basement)
5. Level 4 (Escape Sewer)
6. End Screen
### Classes
- Player Class
- Item Class
- Specific Item Classes (Each Item Has Its Own Class)
- Color Minigame Class
Whackamole Class
- Level Class
